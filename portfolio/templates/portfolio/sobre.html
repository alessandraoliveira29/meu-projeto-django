📚 Apresentação da Aplicação Portfolio

🚀 Tecnologias Utilizadas

  • Backend: Python, Django
  • Frontend: HTML5, CSS3, Templates Django
  • Base de Dados: MySQL
  • Serviços Externos: PythonAnywhere para hospedagem, GitHub para versionamento de código

🏛️ Arquitetura da Aplicação

A aplicação segue o padrão MVT (Model-View-Template) do Django:

  • Model: Representa os dados (Disciplina, Projeto, Tecnologia, etc.)
  • View: Funções em Python que controlam o fluxo de dados
  • Template: Ficheiros HTML que renderizam o conteúdo para o utilizador

Componentes principais:

  • Base de Dados MySQL
  • Servidor Backend Django
  • Frontend com HTML e CSS
  • Serviços externos: PythonAnywhere, GitHub

Desenho da Arquitetura:

📊 Modelação da Base de Dados

A base de dados inclui as seguintes entidades principais:

  • Disciplina — relacionada com Projeto (OneToOne)
  • Projeto — relacionado com várias Tecnologias (ManyToMany)
  • Projeto — relacionado com várias Imagens (OneToMany)
  • Projeto — relacionado com Detalhes Técnicos (OneToOne)
